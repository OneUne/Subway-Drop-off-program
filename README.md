# 저 이번에 내려요
2019년 1학기 공학설계 수업 중 진행하였던 '지하철 하차 알리미' 아두이노 구현 프로젝트입니다.
저는 본 프로젝트에서 블루투스와 거리센서, 서보 모터 등을 통해 지하철 스크린 도어 및 잔여 좌석수를 표현하는 일을 담당하였습니다.
